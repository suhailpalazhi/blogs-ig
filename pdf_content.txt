📝  Project  Title  
Campus  Creatives  –  A  Digital  Magazine  for  Students  to  Share  Thoughts  and  Art  
 
🎯  Project  Objective  
Build  a  web  platform  where  students  can  share  their  creative  works—like  writings,  drawings,  or  
photography—similar
 
to
 
an
 
online
 
magazine.
 
 
Visitors
 
can
 
read
 
posts,
 
but
 
only
 
registered
 
users
 
can
 
create
 
posts,
 
like,
 
and
 
comment.
 
 
🏗  Tech  Stack  
●  Backend :  Python  Django  (with  Django  REST  Framework  for  APIs)  
 ●  Frontend :  Next.js  (React-based  framework)  
 ●  Database :  PostgreSQL  (or  SQLite  for  development)  
 ●  Authentication :  JWT  or  Django’s  built-in  authentication  with  REST  API  tokens  
 ●  Deployment  (optional) :  Vercel  for  frontend,  pythonanywhere  for  backend  
 
 
🔑  Core  Features  
User  Roles  
1.  Public  (unauthenticated) :  
 ○  View  posts  and  comments  only.  
 2.  Authenticated  users  (students) :  
 ○  Create  posts  (with  text,  image,  or  artwork  uploads)  
 
○  Like  and  comment  on  posts  
 ○  Edit  or  delete  their  own  posts  and  comments  
 
Functional  Requirements  
●  User  registration  and  login  
 ●  Profile  management  (name,  bio,  avatar)  
 ●  Post  creation  with  optional  image/art  upload  
 ●  Like  and  comment  system  
 ●  Paginated  feed  of  posts  
 ●  Responsive  design  for  mobile  and  desktop  
 ●  Search  or  filter  posts  by  category/tags  
 ●  Admin  panel  for  moderation  
 ●  Dark  mode  toggle   
 ●  Notifications  for  likes/comments  (optional)   
 
 
✅  Deliverables  
●  GitHub  repository  with  README.md  explaining  setup  steps  
 ●  Working  demo  link  
 ●  Presentation/demo  of  features  at  the  end  of  the  project
